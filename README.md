# ghidra-bridge

Atomic plugin exposing only `ghidra_bridge`.

## Tool

- `ghidra_bridge`

## Environment

- `GHIDRA_BRIDGE_URL` (default: `http://127.0.0.1:18080`)
- `PLUGIN_GHIDRA_BRIDGE_ENABLED` / `PLUGINS_GHIDRA_BRIDGE_ENABLED`

## Notes

- Endpoint is restricted to loopback hosts (`127.0.0.1` / `localhost` / `::1`).
